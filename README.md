# Yoga Posture ML

A Yoga Pose Classification and Feedback system built with an agent-based architecture.
This project detects body landmarks using MediaPipe, extracts pose features, trains a Random Forest model, and provides real-time pose classification with feedback through a Gradio interface.

## Supported Poses
The project currently uses these pose categories:
- `Tree`
- `warrior2`
- `goddess`
- `downdog`
- `plank`

## Project Structure
- `app.py` - Gradio app entry point for webcam/image input, pose detection, classification, and feedback.
- `train.py` - Training pipeline that extracts pose features, trains a Random Forest classifier, evaluates accuracy, and saves the model.
- `requirements.txt` - Python dependencies.
- `dataset/train/` and `dataset/test/` - Image folders organized by pose class.
- `model/pose_landmarker_lite.task` - MediaPipe pose landmark model asset.
- `model/model.pkl` - Saved trained classifier produced by `train.py`.
- `agents/` - Modular agents for input processing, pose detection, feature extraction, classification, and feedback.

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Prepare your dataset:
   Organize images into class folders inside `dataset/train/` and `dataset/test/`.
   Example structure:
   ```text
   dataset/
     train/
       Tree/
       warrior2/
       goddess/
       downdog/
       plank/
     test/
       Tree/
       warrior2/
       goddess/
       downdog/
       plank/
   ```

3. Train the model:
   ```bash
   python train.py
   ```
   This will create `model/model.pkl` and print evaluation accuracy.

4. Run the Gradio app:
   ```bash
   python app.py
   ```

## Usage
- Upload an image or enable webcam input in the Gradio interface.
- The app detects body landmarks, classifies the pose, and displays a confidence score.
- It also returns pose-specific guidance or correction feedback.

## Notes
- If no `dataset/test/` images are found, `train.py` automatically splits the training data for validation.
- The classifier uses feature vectors generated from MediaPipe pose landmarks.
- Feedback is generated by `agents/feedback_agent.py` based on predicted pose and confidence.

## Requirements
The project depends on:
- `opencv-python`
- `mediapipe`
- `scikit-learn`
- `gradio`
- `joblib`
- `numpy`

## Agent Components
- `InputAgent` - standardizes uploaded images and webcam frames.
- `PoseDetectionAgent` - detects pose landmarks with MediaPipe.
- `FeatureExtractionAgent` - converts landmarks into a flattened feature vector.
- `ClassificationAgent` - loads the trained model and predicts the pose.
- `FeedbackAgent` - returns helpful posture feedback based on prediction.
